# crm-h5

说明： 
	1.本项目使用了，VantUI框架
	2.项目借助postcss-plugin-px2rem 插件，将项目中的px，转转成
		rem（设计稿中，量多少就写多少px）。
	3.简单封装了 axios,并进行接口的统一管理,最后把封装好的接口挂载到了
		Vue原型上。
